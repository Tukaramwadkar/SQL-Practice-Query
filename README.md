# 📘 SQL Practice Assignment

## 📌 Description

This repository contains my **SQL practice assignments** completed as part of learning **Database Management Systems (DBMS)**.

The purpose of this assignment is to understand and practice different types of SQL queries including basic, intermediate, and advanced concepts.

---

## 📚 Topics Covered

* Basic SQL Queries (SELECT, WHERE)
* Filtering Data (AND, OR, NOT)
* Sorting (ORDER BY)
* Aggregate Functions (COUNT, SUM, AVG)
* Grouping (GROUP BY, HAVING)
* Joins (INNER JOIN, LEFT JOIN, RIGHT JOIN)
* Subqueries

---

## 📝 Assignment Details

The assignment includes multiple questions such as:

* Retrieve employees based on salary conditions
* Find records using BETWEEN, IN, LIKE
* Perform queries using AND / OR conditions
* Work with NULL values
* Apply aggregate functions on data
* Perform joins on multiple tables
